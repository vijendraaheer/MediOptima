# 🏥 MediOptima - Smart Hospital Management System

MediOptima is an AI-powered hospital management system that helps optimize hospital operations using data analytics and machine learning.

---

## 🚀 Features

- 📊 **Interactive Dashboard** for hospital analytics  
- 🛏️ **Bed Requirement Prediction** using ML models  
- 👨‍⚕️ **Staff Optimization System** for efficient workforce allocation  
- 📈 **Time-Series Forecasting** for patient inflow  
- 🚨 **Surge & Outbreak Detection** with alert system  
- 🤖 **AI-Based Insights Generator** for operational intelligence  
- 🔐 **Role-Based Authentication System** (Admin / Doctor / Staff)
